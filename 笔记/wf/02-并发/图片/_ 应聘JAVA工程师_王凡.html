已工作1年以上可不写描述，在读或应届生推荐填写以下3部分内容：

 

GPA：

通常填写两位数，并且把总分写出来（也可以四舍五入写一位数），成绩优异可以描述名次。尚未毕业则写当前已知的GPA。

如果GPA较低，可以计算一个"专业GPA"，如果GPA低于3.0可以不写。

写法1：GPA：3.76 / 4.00

写法2：GPA：3.8 / 4.0

写法3：GPA：3.86 / 4.00（专业前5%）

写法4：GPA：3.0（专业GPA：3.6）

 

荣誉奖项：

挑选几个最有含金量的写即可，不要重复罗列相同的奖项，多次获得的奖项可以合并。 

写法1：荣誉奖项：校先进个人（2010-2014）、学生会优秀干部（2012-2015）

写法2：荣誉奖项：市级优秀毕业生（专业前3%）、一等奖学金（2011，2013）

 

相关课程：

应届求职或找实习：如果经历不多，内容不够一页纸，适当添加相关课程，建议不要超过1行，无需写每课成绩。

已工作跳槽：建议不写课程。如有1段以上教育经历，只写最高学历的、跟岗位相关的课程即可。

考研或学术：以重要性和成绩罗列跟申请专业/科研项目相关的课程，可写成绩，挑选填写不用把所有课都列出。

写法1（求职）：相关课程：财务建模与报表分析、公司金融、投资银行学

写法2（学术）：相关课程：原子物理学 (92)、数学物理方法 (92)、理论力学 (90)、固体物理学 (88)、结构和物性 (87)、计算物理学入门 (86)

自定义模块还可以写什么？

自定义模块是可以根据此模块的内容按照自己的需求填写的，除了个人信息，教育信息，工作信息等，你还可以填入：

  • 作品展示：如果有作品也可以单独列一个模块放出作品链接，不过一般不建议把作品内容直接插入简历中，建议单独一份文件附在简历后面一起投递，或者直接放链接即可。
  • 求职意向：一般可以写在简历文件名称或者邮件名称中，在简历里也可以在基本信息中体现，如果希望更明显的可以在自定义模块单独列出来。
自定义模块中的时间一定要填吗？

自定义模块可以自由选择是否填写时间，如果此模块不用于写具体的经历，而是总结性文字则可以不写时间。

教育、工作/实习、活动、项目、奖项等内容都要写上时间，如果是一个时间点或者不到1个月的经历，只写结束时间就可以了，不要出现 9月-9月 这样的写法。

自定义模块中职位一定要写吗？

自定义模块是可以根据此模块的内容按照自己的需求填写的，任何条目都可以不写。

如果是写工作/实习、活动、项目等内容都要写清楚你的职位或者你的角色，你主要负责什么样的工作，这个在简历中是非常重要的必填项。
